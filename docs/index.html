起卦流程

1. 静心 · 按住呼吸圆 3 秒，摒弃杂念
2. 选择 · 左滑“随机取数”，右滑“当前时辰”
3. 跳动 · 六位弹跳动画呈现抽取过程
4. 解读 · 结果含义渐入显示，点击圆形可再次占卜

小六壬六神含义解读

大安
平稳守成 · 贵人护持
  • 关键词：稳固、积累、按部就班
  • 建议：把握基础建设与长线规划，宜守不宜躁
留连
反复拖延 · 需耐心
  • 关键词：牵挂、滞后、等待
  • 建议：先排除干扰、理清流程，再推进关键事项
速喜
喜讯来临 · 快速达成
  • 关键词：捷报、机遇、行动力
  • 建议：果断执行，及时收束成果，避免过度冒进
赤口
口舌是非 · 谨言慎行
  • 关键词：争执、火气、分歧
  • 建议：沟通放缓、重复确认，必要时先冷处理
小吉
缓步上扬 · 积小成多
  • 关键词：渐进、细水长流、稳健
  • 建议：持续堆叠小成果，注重节奏与品质
空亡
暂缓观望 · 重新整顿
  • 关键词：停滞、分散、缺口
  • 建议：减少大动作，先补充资源或准备替代方案
